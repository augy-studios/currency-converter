Currency Exchange API

Why self-hosted?

This instance runs the open-source Frankfurter project on infrastructure Augy controls. Being self-hosted doesn't make it faster than the public API, since it's still the same underlying provider data, refreshed on the same daily schedule.

Curious about uptime, update frequency, or provider quirks? See the FAQ & Operational Notes.
